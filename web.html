calculator

This is my first major project. I made it in February 2017. The program was written with a view to the competition at the State Higher Vocational School in Nysa where I took the fourth place. The project consists of two parts, a simple calculator and graphs of mathematical functions.

The calculator contains a number of basic mathematical activities Ie.

  • addition
  • subtraction
  • division
  • multiplication
  • square root
  • percent

The data can be entered using the input field or keyboard located on the page. For this project I used html, css, java script. The second part allow you to draw graphs of such matematik functions as:

  • linear function
  • Quadratic function
  • Rational function
  • Sine function
  • Cosine function
  • Tangent function
  • Cotangent function

Most functions can be called with different parameters. I use java script, canvas to generate function graphs
